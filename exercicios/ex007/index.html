principais formatações

negrito / destaque

nesta frase temos um termo em negrito utilizando a tag < b > ( sentido nao semantico)

nesta frase temos um termo em destaque usando a tag < strong > (sentido semantico)

italico / ênfase

nesta frase temos um termo em italico usando a tag "i" (não semantica)

nesta frase temos um termo em italico usando a tag "em" (sentido semantico)

texto marcado

podemos tambem criar um texto marcado usando a tag < mark >

texto grande e pequeno

estamos criando um texto grande e um texto pequeno

texto deletado

podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado

texto inserido

podemos marcar um texto como inserido para dar enfase e indicar que ele foi adicionado depois.

texto sobreescrito

para inserir coisas do tipo x10, basta usar a tag < sup >

texto subescrito

para inserir coisas do tipo H2O, basta usar a tag < sub >
